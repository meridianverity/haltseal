# Patent and IP posture

This repository is a public-safe review artifact. It does not include a claim chart, implementation map, legal opinion, production SDK, commercial terms, field-of-use terms, or patent-license grant.

Publicly describable themes:

- fail-closed runtime action control
- permit-before-action posture
- short-lived audience-bound permits
- signed-head freshness and continuity posture
- nonce plus monotonic anti-replay
- evidence-only audit records
- structured HOLD and DENY dispositions

The repository is designed to create technical clarity while preserving written-scope depth.

For current patent status, assignments, maintenance, terminal disclaimers, and licensing, consult official records and counsel.
